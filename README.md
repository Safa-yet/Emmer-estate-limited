# 🧱 Emmer-Estate-Limited

A modern and responsive construction company landing page built using Tailwind CSS. This project showcases a professional UI design for a construction/developer company website.

---

## 📖 Project Overview

 Emmer-Estate-Limited  is a clean and modern landing page designed for a construction or real estate development company. It highlights services, company information, and provides a strong visual experience for potential clients.
